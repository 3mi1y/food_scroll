***********************************************
DIRECTORY STRUCTURE
***********************************************

views
This is where the actual web pages live.
	- there is a file called unused within the views folder, this is for experimental php pages that aren't officially a part of the system, but that may have been created as part of learning or they are leftover from whatever and not being used

app_code
this is where our 'in the middle' or 'behind the scenes code lives. This code is used to access the database, do stuff, etc.

resources
this is where things like images, css, js and bootstrap live

includes
this is php templating material. Example, the master page

documentation
tutorials on setting up workspace
ER Diagram
Relational Schema
database scripts
